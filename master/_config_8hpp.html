Detailed Description

Central configuration: pin assignments, physical constants, and PID gains.

All hardware pin numbers, drivetrain geometry, miner/shooter tuning values, and PID constants live here so that changes only need to be made in one place. Every subsystem and autonomous step pulls its constants from this file via its includes.
