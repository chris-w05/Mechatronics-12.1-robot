bool Robot::parseCmdWithUpToTwoFloats ( const char * cmdStr,
char * outCmd,
float & outP1,
bool & outP1Valid,
float & outP2,
bool & outP2Valid )
inlineprivate

Parse a keyword + up to two float parameters from a command string.

Parse a command string of the form <keyword> [float] [float].

Parameters
cmdStrInput string.
outCmdOutput buffer for the keyword (size >= CMD_TOKEN_MAX).
outP1First numeric parameter output.
outP1ValidTrue if the first parameter was parsed.
outP2Second numeric parameter output.
outP2ValidTrue if the second parameter was parsed.
Returns
true if at least a keyword was extracted.

Keywords may be multi-character (up to CMD_TOKEN_MAX-1 chars). Delimiters between tokens may be spaces, commas, colons, or tabs.

Returns
true if at least the command keyword was found.
