Detailed Description

Ramsete nonlinear tracking controller for differential-drive robots.

Given a reference pose and feedforward chassis velocities, it produces corrected linear and angular velocity commands that drive the robot toward a reference trajectory while compensating for pose error.

Reference: Ramsete — A non-linear time-varying feedback controller for mobile robots on a sampled-data basis (Banni, Brandin, Sciavicco, 1996).
