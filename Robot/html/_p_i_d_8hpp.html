Detailed Description

PID controller classes with optional nonlinear extensions and feedforward support.

Two classes are provided:

  • PID — A lightweight base controller with proportional, integral, and derivative terms. Derivative is supplied externally (from an encoder velocity signal, for example).
  • PIDController — Extends PID with anti-windup, an IIR output filter, and optional nonlinear gain functions / feedforward callables for advanced control schemes.

Function-pointer typedefs (FeedforwardFn, DerivativeFeedforwardFn, KpFn, etc.) allow the caller to inject arbitrary nonlinear behaviours without subclassing.
